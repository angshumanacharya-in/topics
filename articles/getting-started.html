Getting Started

Prerequisites

Setting up your environment requires zero complex build dependencies. Because the core engine runs on vanilla HTML5, CSS3, and JavaScript, you can host these files on any static web server or GitHub Pages.

  1. A local HTTP development server (Node.js Express, Live Server, Python http.server, or Nginx).
  2. Basic understanding of HTML markup for writing topics.
  3. A modern web browser with JavaScript enabled.

Testing Checklist

Verification Steps
  • Click between sidebar topics and verify clean URL paths like /articles/getting-started.
  • Press F5 on any URL route to confirm the SPA fallback resolves without 404 crashes.
  • Click the top-left logo to fold and unfold the dark sidebar drawer.
